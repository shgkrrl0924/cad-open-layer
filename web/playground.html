HOW IT WORKS

DXF bytes go into extractFloorplanFromDxf(), a wasm-bindgen export. The Rust pipeline runs end-to-end in your browser — parse → 5 extraction algorithms → semantic floorplan. Walls render from the centerline polylines. Nothing leaves your machine.

THE SAME CODE IN RUST

use cad_dxf_parser::parse_all;
use cad_extract::extract_floorplan;

let doc = parse_all(&bytes)?;
let plan = extract_floorplan(&doc.entities)?;
